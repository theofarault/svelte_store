# Projet magasin en svelte  

Réalisé par Mathis Boucher et Théo Farault

## Lancer le projet 

Cloner le projet

puis  

```bash
# Installer les dependances
npm install   

# preparer le projet
npm run prepare

# Lancer le projet
npm run dev  
```
